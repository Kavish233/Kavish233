# 💫 About Me:
Tweedejaars Software Development student aan het Grafisch Lyceum Rotterdam.<br>Ik ben momenteel op zoek naar een stageplek en mijn passie ligt vooral bij front-end development, waarbij ik graag werk aan gebruiksvriendelijke en creatieve webinterfaces.
